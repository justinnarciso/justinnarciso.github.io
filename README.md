# justinnarciso.github.io

My marketing portfolio site, hosted on GitHub Pages.

## Blog

Blog posts are stored in `posts.json` and link out to Substack. To add a new post, just add an entry like this:

```json
{
  "title": "Your Post Title",
  "date": "2026-04-11",
  "excerpt": "Short description for the card.",
  "url": "https://justinnarciso.substack.com/p/your-post-slug"
}
```

Posts show up newest first. If there are none, the page just says "Coming soon."
